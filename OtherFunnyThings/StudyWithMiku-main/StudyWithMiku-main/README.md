- [x] Tips设置

- [x] 在线人数

## 部署TIPS

### Umami

自行部署的话建议将 `<head>` 中的 Umami 统计代码更改为自行部署的统计器

```
<!-- Umami START -->
  <!-- 更改为你自己部署的Umami统计器 -->
<!-- Umami END -->
```

也别忘了去修改`mikuplayer`中的**Token**、**URL**值
